# 🧮 Calculadora Web

Calculadora funcional desenvolvida com HTML5, CSS3 e JavaScript puro, com suporte às quatro operações matemáticas básicas e interface limpa e responsiva.

## ✨ Funcionalidades

- Operações: adição, subtração, multiplicação e divisão
- Limpeza de tela (AC)
- Interface responsiva
- Feedback visual nos botões

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
